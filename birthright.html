Overview

Birthright is a competitive 1v1 strategy card game built around a constant tug-of-war for advantage. Matches come down to strategy, resource management, and utility usage, as both players trade momentum back and forth trying to outplay each other in a drawn-out war of attrition.

Now released and playable, Birthright has a small but growing playerbase, 30 downloads and players so far, and climbing.

Built With

The client is built in Godot using C#, with a custom Rust module running on SpacetimeDB handling game state and turn logic for real-time online multiplayer. The split keeps both players perfectly in sync with the server, with moves and board state updating instantly as they resolve.
